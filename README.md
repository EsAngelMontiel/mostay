# README #

Sitio Web de Mostay.

### What is this repository for? ###

* Desarrollo del Sitio web de Mostay
* 1.0

### How do I get set up? ###

#### Instalación
```bash
npm install
```

#### Desarrollo
```bash
# Modo desarrollo (watch)
npm run dev
# o
./dev.sh start

# Compilación manual
npm run build
# o
./dev.sh build
```

#### Tecnologías
* Bootstrap 4.4.1
* Sass (Dart Sass)
* Terser (JavaScript minification)
* Chokidar (File watching)
